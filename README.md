# hoop rush v0.8
Hoop Rush is a fast-paced hyper-casual arcade game where timing is everything.



Tap, click, or press space to jump the ball through moving hoops. Chain perfect shots to increase your score and test your reflexes as the game speeds up. One mistake and it’s game over!



Easy to play, hard to master.

How many perfect hoops can you score?



Tap / Click / Spacebar –> Jump
